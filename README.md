# MNIST PyTorch

Handwritten digit recognition neural network trained on the MNIST dataset using PyTorch.

Requirements.
* Python
* PyTorch
```
conda install pytorch torchvision torchaudio cudatoolkit=10.2 -c pytorch
```

Usage.
```
python train.py
python test.py
```

Evaluation metrics.
```
Accuracy: 9919/10000 (99.19%)
```
